# 🌍 Stamped — World Travel Tracker

Android application for tracking visited countries around the world.

## Features
- 🗺️ Interactive world map with clickable countries
- ✅ Mark countries as visited
- 📊 Statistics (visited count, % of world, continents)
- 🔐 Firebase Authentication (Anonymous, Email, Google)
- ☁️ Firebase Firestore sync
- 🔔 Firebase Push Notifications
- 💾 Room local database (offline support)
- 🌐 Internationalization (English + Macedonian)
- 📱 Portrait + Landscape layouts
- 📋 Countries list with search

## Tech Stack
- Kotlin
- Firebase (Auth, Firestore, Messaging, Analytics)
- Room Database
- WebView + JavaScript Bridge
- Material Design 3
